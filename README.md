H5P Course Presentation
==========

Combine texts, multimedia and tasks into a presentation. Test

[See it in action on H5P.org](https://h5p.org/presentation)

## Getting started

Install all the dependencies:

```bash
npm install
```

Build project for production:

```bash
npm run build
```

Build continuously

```bash
npm run watch
```

Run tests:

```bash
npm test
```
